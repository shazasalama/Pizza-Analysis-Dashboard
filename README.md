# Pizza Sales Analysis Dashboard

## Project Overview
This dashboard analyzes pizza sales data to understand customer ordering patterns, product performance, and sales distribution across categories and sizes.

## Tools Used
- Excel

## Key Metrics
- Total Orders
- Total Sales
- Average Order Value
- Most Ordered Pizza

## Key Insights
- Identified the most popular pizza types
- Analyzed sales distribution by category
- Examined hourly ordering patterns
- Compared pizza size sales distribution

## Dashboard Features
- Orders trend analysis
- Top 10 selling pizzas
- Sales by pizza category
- Orders by hour of the day
